RAINPASS Mobile App

RAINPASS Mobile App

RAINPASS를 스마트 폰 앱에 탑재

RAINPASS Mobile App에서 생성되는 RAINPASS ID와 기존 시스템을 연계하여 유져 인증 등 다양한 용도로 활용 할 수 있습니다.

기능

Bar Code표시

RAINPASS ID를 바코드로 표시
바코드 스캐너로 RAINPASS ID를 읽어들여 시스템과 연계하므로서 다양한 활용이 가능합니다.

QR Code표시

RAINPASS ID를 QR Code로 표시
QR Code 스캐너로 RAINPASS ID를 읽어들여 시스템과 연계하므로서 다양한 활용이 가능합니다.

QR코드 연계

QR코드 정보와 RAINPASS ID를 시스템과 연계해 다양한 활용이 가능합니다.

모바일 브라우져 연계

모바일 브라우져에 RAINPASS ID를 넘겨줌으로써 여러 활용이 가능합니다.

설치 사용

  1. App Store나 Google Play에서 'RAINPASS'로 검색하여 앱을 인스톨합니다.
  2. 사용하고자 하는 서비스의 안내에 따라 등록을 합니다.
    *서비스에 따라 메일로 발송 되는 등 방법이 다를 수 있습니다.

PC 사용시

  1. App 왼쪽 상단의 버튼을 선택
  2. PC화면에 표시된 QR코드를 촬영
  3. PIN코드* 입력
  4. 리스트에 추가한 서비스명이 표시
  5. 서비스명을 선택하면 RAINPASS ID가 표시됩니다.

스마트폰 사용시

  1. RAINPASS APP Connect 버튼 선택
  2. PIN코드* 입력
  3. 리스트에 추가한 서비스명이 표시
  4. 서비스명을 선택하면 RAINPASS ID가 표시됩니다.

*PIN코드는 서비스와 RAINPASS Mobile App연결 시에 비밀번호와 같은 역할을 합니다.

RAINPASS 활용

RAINPASS Smart Card

RAINPASS를 Smart Card에 탑재
Smart Card에서 RAINPASS ID를 생성합니다.

상세

RAINPASS SSH

RAINPASS를 이용한 ssh로그인 관리
유져가 바뀔 때마다 패스워드나 공개키를 따로 관리하지 않아도 됩니다.

상세

RAINPASS Encryption

Encrypt과 Decrypt 용 시크릿 키에 RAINPASS ID를 활용.
