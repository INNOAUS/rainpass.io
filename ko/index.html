보다 안전하고 간단한 신개념 예측 불가능 소멸성 ID

ID는 변하지 않는 것이라는 고정관념에 잡혀있지 않습니까?
ID가 주기적으로 변한다면 정말 많은 응용이 가능합니다.
무한한 가능성을 가진 RAINPASS를 통해 놀라운 경험을 체험해 보세요.

RAINPASS 란

RAINPASS는 오프라인 상에서 시스템간의 일정한 간격으로 무작위하고 고유한 값(ID)을 실시간으로 동시에 생성하는 기술입니다.

RAINPASS Technology

RAINPASS 특징

통신이 불필요

ID생성을 위해 시스템 간에 데이터 교환이 필요 없기 때문에 오프라인 상에서 동작합니다.

No Password

ID가 고정되어 있지 않기 때문에 패스워드가 필요 없습니다.

경량

가볍습니다.(RAINPASS SmartCard에 내장된 알고리즘 크기는 6KB)

자릿수 조정

2자리부터 2048자리까지 설정 가능

생성 주기 설정

RAINPASS ID의 생성 주기 설정이 가능합니다.(30초, 45초 등)

문자 조합

RAINPASS ID를 숫자, 영문 대소문자, 특수문자<()~@#$%&()+-[]{}./<=>?"; 조합이 가능합니다.

RAINPASS ID 동시 생성

서로 다른 문자 길이의 RAINPASS ID를 동시에 생성하므로 백그라운드 시스템 인증(TOKEN) 등에 이용할 수 있습니다.

다양한 언어

C, C++, C#, java등 거의 모든 언어로 만들 수 있습니다.

CPU, MCU

CPU, MCU가 탑재된 모든 시스템에 적용 가능합니다.

다양한 용도로 활용을 할 수 있습니다.

RAINPASS Mobile App

RAINPASS 대표적인 예시 및 장점

RAINPASS Mobile App과 함께 시스템의 사용자 인증 분야에 이용합니다.

사용자는 Mobile App에 표시되는 RAINPASS ID를 입력하는 것만으로 인증이 가능합니다.

그러므로

  • 사용자는 ID,패스워드를 기억할 필요가 없습니다.
  • ID가 일정한 간격으로 변화하므로 패스워드가 필요 없습니다.
  • ID,패스워드 유출에 대한 걱정이 없습니다.
  • 패스워드 돌려쓰기로 인한 피해를 막을 수 있습니다.
  • OTP가 필요 없습니다.
  • 외부 공격으로부터 보다 안전해집니다.
